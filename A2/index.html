Assignment 2

Introduction

It is time to show case your Javascript skills with d3.js! In this assignment you will apply your knowledge from the previous assignment about visual encoding. You will also be required to implement, using d3.js, two well-known multi-attributes visualization techniques, namely the scatter plot matrix and the parallel coordinates which have been introduted in the lecture. Overall, your tasks are:

  1. Load the dataset
  2. Decide on 3 numerical attributes and 1 categorical attributes from the given dataset you would to display!
  3. Create the attribute labels.
  4. Create the scatter plot matrix of those 4 attributes
  5. Create the parallel coordinates plot of those 4 attributes
For this exercise, all your solution must be implemented by yourself in index.js script. Your solution will appear under div#visualization element in the index.html.

Task 1: Prepare your data (10%)

To start, you have to prepare your data and label before implementing the visualization technique. In the global scope of the index.js you are tasked with:

  1. Loading the dataset.
  2. Defining your 3 numerical and 1 categorical attributes
  3. Defining the color mapping/scheme you are using for labeling your data points and lines in both plots
Hint: Take a quick look into the dataset and decide what kind of information that you want to convey with both the Scatter Plot Matrix and the Parallel Coordinates! Regarding coloration, take a look at d3.scaleOrdinal.

Task 2: Create color legend (5%)

When using color in visualizing a dataset, it is crucial to convey the color mapping to the users of the visualization. In this function you need to make a consistent color labelling, so that viewers can associate data with color within and across multiple plots. Implement your legend in createLegend() function!

Hints: Use the color mapping/scheme you defined before to serve as the visual channel for the label.

Task 3: Create the scatter plot matrix for each numerical attributes pair (25%)

In a scatter plot matrix, each attribute is assigned to one column and one row of a matrix such that each cell contains a combination of two attributes that are then depicted by a scatter plot. A helper function createScatterPlotMatrix function is provided to create 3 x 3 matrix space for scatter plots for each pair of numerical attributes. You are required to express the pairwise combination attribute value of the data on each cell. In this task, you need to:

  1. Fill in the correct input domain for the scales
  2. Show the points for all the data accordingly
  3. Color the points according to your color scheme
Write your implementation in the scatterPlot function!

Hints: there is the d3.extent function to find value ranges.

Task 4: Create horizontal parallel coordinates (35%)

Realize the parallel coordinate plot with the same attributes as the scatter plot matrix. In contrast to scatter plot matrixes, in a parallel coordinates plot each attribute is represented by a single axis parallel to all others and each data item shows as a polyline instead of a point. You are expected to

  1. Show the vertical axes next after the other horizontally
  2. Label the axes and show the ticks
  3. Produces connected lines for all data
Use the same color scheme as the previous task.

Hints: To implement the parallel coordinates plot, you will need a storage (for example a map) to hold your scales for each axis. The usage of an JavaScript object is not recommended here, since it is not iterable and, therefore, cannot be used to create the axes with the usual join pattern. Another useful tool, provided by d3 is the d3.line function, that -- given an array of [x,y] values -- creates a string for an SVG (cor canvas) path, that can then be used inside a SVG path element's d attribute.

Task 5: Analyze the data (15%)

Scatterplots are good at showing correlations between attributes; a scatterplot matrix allows to explore correlations between all chosen attribute pairs. Additionally, visual cluster analysis can be performed. Parallel coordinates on the other hand are best at showing multi-attribute relations.

Let us draw some information from your plots to learn more about the data:

  1. Find at least one attribute pair that shows a correlation. State which kind of correlation it is and how the corresponding pattern looks like in a scatterplot and in parallel coordinates.
  2. Which attribute or combination of attributes separates the colors from your categorical attribute best?
Write your answer in the file "analysis.md".
